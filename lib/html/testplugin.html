Wie man ein abbozza! plugin erstellt

Diese Datei enthält eine kurze Anleitung, wie man ein abbozza! Plugin erstellt.

Wo gehören plugins hin?

Das Verzeichnis, in dem abbozza! Plugins hinterlegt werden hängt vom jeweiligen System ab.

Arduino

Für den Arduino gibt es zwei mögliche Verzeichnise. Entweder im Sketchbook-Ordner des jeweiligen Nutzers unter <user_sketchbook>/tools/Abbozza/plugins oder im Installations-Verzeichnis der Arduino IDE unter /tools/Abbozza/plugins.

Andere Systeme

Für alle anderen Systme ist es entweder das Verzeichnis <home>/.abbozza/<system>/plugins oder <abbozza_root>/plugins. Dabei wurden die folgenden Platzhalter verwendet:

  • <home>: Das Heimatverzeichnis des Nutzers.
  • <system>: Der Name des jeweiligen Systems, z.B. calliopeC.
  • <abbozza_root>: Das Verzeichnis in dem abbozza! installiert wurde.

Was man auf jeden Fall braucht

In dem jeweiligen Plugin-Verzeichnis muss ein Verzeichnis oder eine Jar-Datei vorliegen, die alle notwendigen Dateien enthält. Im Folgenden gehen wir von einer Verzeichnisstruktur aus. Wie man die Jar-Datei erstellt, wird im Anschluss beschrieben.

In dem Verzeichnis muss eine XML-Datein plugin.xml vorliegen, die mindestens die folgenden Form haben muss: 

<?xml version="1.0" encoding="UTF-8"?>
<plugin id="testplugin">
    <!-- das Attribut id ist ein eindeutiger Identifier des Plugins -->
    
   <!-- Name des Plugins -->
   <name>Test-Plugin</name>
   
   <!-- Beschreibung des Plugins -->
   <description>Dies ist ein einfaches Testplugin</description>
</plugin>
